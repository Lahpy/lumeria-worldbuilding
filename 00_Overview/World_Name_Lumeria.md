# World Name - Lumeria

## Current decision

The current working world name is **Lumeria**.

This replaces **Avalon**, which should now be treated as an old placeholder name and source archive label.

## Why Lumeria fits

Lumeria has the right tone for the current direction:

- It feels soft and JRPG/anime high-fantasy without sounding childish.
- It has a light/magic association through "lume," which fits White Magic and divine magic without being too literal.
- It is broad enough to hold beautiful, dangerous, technological, ruined, and mysterious regions.
- It sounds like a world name rather than only a city, kingdom, or forest.

## How it sounds with the current nations

The current map names work naturally under Lumeria:

- Lumeria
- Asterweald
- E'l Ancalen
- Hazelmour
- Namarion
- Klazen
- Luminor
- Truo Crua
- Astrolia

## Possible issue - Luminor

The only naming issue is that **Lumeria** and **Luminor** share a similar light/lum sound.

This is not an emergency, but it should remain a future naming note. If the overlap starts to feel awkward, Luminor could be renamed later while keeping its function as the art, wealth, coast, diplomacy, and high-culture nation.

Possible replacement directions:

- Lioren
- Lunore
- Auremont
- Solenne
- Valoria
- Aurelia Coast

For now, keep **Luminor** unless the overlap becomes distracting.

## Current phrasing

Use this as the baseline world description:

> Lumeria is a world of ancient forests, luminous kingdoms, ash-scarred frontiers, canal cities, underground republics, star islands, and machine-lit towers. At its heart lies the Asterweald, a forest older than any nation's history and stranger than any school of magic can explain.

## Naming rule going forward

Do not make every major name sound like Lumeria. The world name should feel like the container, not a repeated motif.

Names can still vary by region:

- Asterweald should sound ancient, soft, forested, and mysterious.
- E'l Ancalen should sound elegant, magical, and formal.
- Hazelmour should sound autumnal, harsh, and frontier-like.
- Namarion should sound maritime, wealthy, and mercantile.
- Klazen should sound sharp, engineered, and modern.
- Truo Crua should sound grounded, stone-carved, and subterranean.
- Astrolia should sound celestial and distant.
